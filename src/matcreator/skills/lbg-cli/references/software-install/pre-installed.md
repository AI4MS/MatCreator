# Pre-installed software on the Bohrium image

Source: <https://docs.bohrium.com/docs/userguide/image>

Before installing, run `which <cmd>` or `pip show <pkg>` to check
whether it is already present ‚Äî most common scientific tooling ships
with the image.

## Inventory

- **Python stack** ‚Äî Python 3.10, `pip`, Anaconda, Jupyter Lab.
- **File tools** ‚Äî `wget`, `curl`, `unzip`, `emacs`, `vim`, `rsync`,
  `tree`, `git`.
- **Build toolchain** ‚Äî `cmake`, `build-essential`.
- **System utilities** ‚Äî `htop`, `ncdu`, `net-tools`.
- **DP ecosystem** ‚Äî Lebesgue Utility, DP-Dispatcher, Dpdata.
